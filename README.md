# ConfessItUp 💌

ConfessItUp is a **city-based anonymous confession app**.  
Read, post, and like confessions from your city — no random global feeds, just your local vibe.  

Built for **fun, emotions, and local connections**, MVP-style with speed in mind.  

---

## ⚡ MVP Features

- Anonymous confessions per city  
- Dynamic city routes (`/[city]`) for local feeds  
- Post confessions & like others  
- Fast, responsive UI with **shadcn/ui** + **Tailwind CSS**  
- Supabase backend with **PostgreSQL**, **RLS policies**, and **RPCs**  

---

## 🛠 Tech Stack

- **Frontend:** Next.js 13 (App Router), React 18  
- **UI:** shadcn/ui + Tailwind CSS  
- **Backend:** Supabase (DB + RLS + RPC)  
- **Deployment:** Vercel recommended  
- **Auth:** None for MVP (anonymous)
